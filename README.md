# 🚗 Road Accident Analysis Dashboard

## 📊 Project Overview
This project presents a Road Accident Analysis Dashboard created using Microsoft Excel.

## 📁 Dataset
The dataset contains information about:
- Location
- Accident Type
- Vehicle Type
- Weather Condition
- Severity
- Casualties
- Date

## 📈 Dashboard Features
- Accidents by Location
- Severity Distribution
- Vehicle Type Analysis
- Weather Condition Analysis
- Monthly Trend of Accidents
- Casualties Analysis

## 🎯 Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
